# Simple Calculator with History (Python)

A beginner-friendly Python calculator that performs basic arithmetic operations and includes file-based history management. The calculator allows users to calculate expressions, save results, view previous calculations, and clear the saved history.

## Features

* Perform addition, subtraction, multiplication, and division.
* Supports equations with or without spaces (e.g., `8 + 9` and `8+9`).
* Prevents division by zero.
* Save calculation history to a `history.txt` file.
* View all previously saved calculations.
* Clear the saved history when needed.
* Built using Python functions, loops, file handling, and regular expressions (`re`).

## Concepts Practiced

* Functions
* Conditional Statements (`if`, `elif`, `else`)
* Loops (`while`)
* File Handling (`read`, `write`, `append`)
* String Manipulation
* Regular Expressions (`re.split`)
* User Input Handling
